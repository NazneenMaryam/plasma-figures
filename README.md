# Broadband Laser–Plasma Interaction Figures

This repository contains the Python scripts used to generate the figures for the review paper:

**Controlling Nonlinear Parametric Instabilities in Laser–Plasma Interaction Using Broadband and Low-Coherence Lasers**

## Contents

- Figure 1: Broadband-laser temporal intensity modulation
- Figure 2: Coherence time versus laser bandwidth
- Figure 3: SBS reflectivity comparison
- Figure 4: Conceptual intermittent SRS bursts
- Figure 5: Density-dependent broadband BSRS behavior
- Figure 6: SRS suppression by angular incoherence

## Requirements

- Python 3
- NumPy
- Matplotlib
